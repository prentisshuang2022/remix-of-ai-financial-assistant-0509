# 用户指南

面向财务、审计、业务管理用户。

## 1. 登录与导航

进入系统后，左侧导航包含：

- **驾驶舱** — 当日核心指标
- **报销审核** — 报销单的智能审核
- **应付管理** — 供应商账期与付款
- **应收管理** — 客户账龄与回款
- **风险监控** — 跨模块风险事件

## 2. 报销审核

1. 进入「报销审核」，顶部展示当日 KPI（高风险单、重复票据、AI 通过率等）。
2. 通过顶部筛选器（日期、部门、风险等级、状态）缩小范围。
3. 列表每行展示规则命中、重复票据、AI 建议与建议理由。
4. 点击单据 ID 进入详情，可执行：
   - **通过**：采纳 AI 建议，单据归档
   - **退回**：填写退回原因
   - **转人工**：进入人工复核队列
5. 点击右上「导出」可下载当前筛选结果 CSV。

## 3. 应付 / 应收

- 列表按账龄分桶展示。
- 勾选多条记录后，点击「制定付款计划 / 催收任务」批量处理。
- 行右侧「>」可查看明细与历史记录。

## 4. 风险监控

- 顶部按等级（高/中/低）汇总。
- 点击事件可执行：**确认（ack）**、**关闭（close）**、**升级（escalate）**。

## 5. AI 自定义分析（各模块通用）

每个模块页面顶部均提供「AI 自定义分析」面板：

1. 点击「配置规则」打开右侧抽屉。
2. 在抽屉顶部「用自然语言快速生成」中输入需求，例如：
   - *上月研发部超过 5000 元的报销，按部门汇总并取 Top10*
   - *逾期 60 天以上的供应商，按金额降序*
3. 点击「AI 解析覆盖当前规则」，系统自动生成多条结构化规则。
4. 在规则列表逐条调整：
   - **类型**：时间 / 筛选 / 分组 / 聚合 / 排序 / 限制
   - **字段**：来自当前模块的字段下拉
   - **操作符**：随类型自动切换
   - **值**：可输入数值、日期、枚举或 Top N
5. 点击底部「应用并取数」或主面板「取数并生成结果」。
6. 结果表下方按钮：
   - **下载 CSV**：浏览器直接下载（含 BOM，可被 Excel 正确识别中文）
   - **发送至钉钉**：一键推送到「财务部 · 工作群」（管理员可在后台配置接收群）

详见 [AI 规则配置指南](./ai-rule-analyzer-guide.md)。

## 6. 常见问题

| 问题 | 解决 |
| --- | --- |
| 列表数据不刷新 | 浏览器右上角「刷新」或切换筛选条件 |
| CSV 中文乱码 | 使用 Excel「数据 → 从文本/CSV 导入」选 UTF-8 |
| 钉钉未收到推送 | 联系系统管理员检查机器人 Token / 群 ID |
| AI 解析结果不准确 | 在抽屉中手动调整规则后再「应用并取数」 |
