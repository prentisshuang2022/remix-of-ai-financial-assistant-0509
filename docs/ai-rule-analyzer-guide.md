# AI 规则配置指南

本指南详细说明「AI 自定义分析」组件的使用方式与规则语义。

## 规则类型一览

| 类型 | 用途 | 可选操作符 | 取值示例 |
| --- | --- | --- | --- |
| 时间 | 限定数据时间范围 | `=` | `上月`、`本季度`、`2025-04` |
| 筛选 | 行级过滤 | `>` `≥` `<` `≤` `=` `≠` `包含` `不包含` `属于` | `5000`、`研发部`、`餐费,差旅` |
| 分组 | 聚合维度 | `按` | （取 field） |
| 聚合 | 度量计算 | `合计` `平均` `最大` `最小` `计数` | （取 field） |
| 排序 | 结果排序 | `降序` `升序` | （取 field） |
| 限制 | 限定结果行数 | `Top` | `10`、`20` |

## 执行顺序

```text
时间过滤  →  行级筛选  →  分组  →  聚合  →  排序  →  限制（Top N）
```

不需要的环节可省略；若仅有「时间 + 筛选」，结果即为明细表。

## 编辑流程

1. 点击主面板的 **配置规则** 打开右侧抽屉。
2. 顶部输入自然语言并点击 **AI 解析覆盖当前规则**，或点击下方示例 chip 快速填入。
3. 在「规则列表」逐条编辑：左上角下拉切换类型、删除按钮在右上角。
4. 底部 **添加规则** 按钮按类型颜色区分，点击即可追加。
5. 点击底部 **应用并取数**，抽屉关闭并立即触发取数。

## 与 ERP / 钉钉的关系

- **取数源**：金蝶 ERP（实时）。各模块自动映射到对应主表。
- **导出 CSV**：浏览器直接下载，文件名形如 `报销-AI分析-1715400000000.csv`。
- **推送钉钉**：默认推送至「财务部 · 工作群」，管理员可在后端配置目标群与 @人。

## 给研发：扩展字段

新增可选字段：在对应页面（如 `Expense.tsx`）修改：

```ts
const ANALYZER_FIELDS = ["部门", "金额", "员工", "日期", "类别", "新字段"];
```

新增预置场景：在 `SCENARIOS` 数组里追加 `AnalyzerScenario`，`match` 中填入触发关键词。
